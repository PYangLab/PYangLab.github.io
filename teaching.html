Phosphoproteomics data analysis

  • PhosR package repository at Github (https://github.com/PYangLab/PhosR)
    PhosR is an R package designed for phosphoproteomics data analysis. The PhosR repo provides the installation of the package and contains examples and tutorials for common analysis steps performed for phosphoproteomics datasets. For a more structured tutorial on PhosR package, please refer to BioinfoSummer 2019 below. <\li>
  • PhosR tutorial in BioinfoSummer 2019 (https://sydneybiox.github.io/BIS2019_PhosR/index.html)
    More details of BioinfoSummer 2019 PhosR tutorial can be found here: https://bis.amsi.org.au/speakers/pengyi-yang/ <\li>
