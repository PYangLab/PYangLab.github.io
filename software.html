Software


  • KinasePA (http://shiny.maths.usyd.edu.au/KinasePA)
    Kinase perturbation analysis (KinasePA) is a web tool that allows you to identify key kinases that are perturbed in two treatments compared to a control condition.

    The input data should be a csv file separated by comma. The rows of the data file are phosphorylation sites and the columns are treatment1 vs control and treatment2 vs control. The values of the data file should be log2 fold changes. Please download this file as an example for required data format: [exampleData] (hint: right click and save)

    Note that the same data is loaded into KinasePA when it is initiated for demonstrating its function.
